### 🚑 Hotfix V7.7.2 - 설치 스크립트 치명적 오류 수정

**1. `install.sh` 설정 적용 실패 문제 해결**
- V7.7.1 배포 후, 사용자의 PC에 `.bashrc` 또는 `.bash_profile` 파일이 아예 없는 초기 환경에서 **설치 스크립트가 아무런 설정도 추가하지 않는 문제**가 발견되었습니다.
- 기존 로직이 `if [ -f "$rc_file" ]`로 파일 존재 여부만 확인하고, 없으면 생성을 하지 않아 발생한 문제입니다.
- **수정 사항**: 
  - 설정 파일이 없으면 `touch`로 생성하도록 변경했습니다.
  - Windows Git Bash 환경에서 `.bash_profile`이 없으면 생성하고, `.bashrc`를 호출하는 로직(`if [ -f ~/.bashrc ]; then . ~/.bashrc; fi`)을 자동으로 추가하여 셸 호환성을 보장했습니다.
